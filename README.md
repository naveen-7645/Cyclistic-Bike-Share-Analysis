\## 📌 Project Overview

This project is part of the Google Data Analytics Capstone. The objective is to analyze Cyclistic bike-share data and identify how casual riders and members use bikes differently.



The goal is to provide insights and recommendations to convert casual riders into annual members.



\---



\## ❓ Business Question

How do casual riders and members use Cyclistic bikes differently?



\---



\## 📊 Dataset

\- Source: Cyclistic Historical Trip Data (Motivate International Inc.)

\- Duration: 12 months

\- Total records analyzed: \~1,048,575 rides

\- Key columns:

&#x20; - ride\_id

&#x20; - rideable\_type

&#x20; - started\_at / ended\_at

&#x20; - member\_casual



\---



\## 🛠 Tools Used

\- Microsoft Excel

\- Power Query

\- Pivot Tables

\- Data Visualization (Charts)



\---



\## ⚙️ Data Processing

\- Combined multiple datasets using Power Query

\- Cleaned and transformed data

\- Created new features:

&#x20; - `ride\_length` (duration)

&#x20; - `ride\_length\_min` (minutes)

&#x20; - `day\_of\_week`

\- Removed errors and handled missing values



\---



\## 📈 Analysis \& Insights



\### 🔹 Average Ride Duration

\- Casual Users: \~23.6 minutes

\- Members: \~12 minutes

👉 Casual users take longer rides (leisure behavior)



\---



\### 🔹 Ride Frequency

\- Members: \~65%

\- Casual Users: \~35%

👉 Members are more frequent users



\---



\### 🔹 Weekly Patterns

\- Members → Weekdays

\- Casual Users → Weekends

👉 Clear difference between commuting vs recreational usage



\---



\## 📊 Visualizations

\- Average Ride Duration by User Type

\- Total Rides by User Type

\- Daily Ride Distribution by User Type



(All charts available in the Excel file and presentation)



\---



\## 💡 Key Recommendations

\- Offer weekend promotions for casual users

\- Highlight cost benefits of annual membership

\- Run targeted marketing campaigns during weekends

\- Encourage frequent riders to switch to membership plans
